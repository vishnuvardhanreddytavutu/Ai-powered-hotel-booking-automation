## 🚀 Overview

This project is an AI-powered hotel room booking automation system built using **n8n workflow automation**. It demonstrates how automation workflows can handle booking requests, store data, and send real-time notifications without manual effort.

---

## 🧠 Features

* Automated hotel room booking workflow  
* Trigger-based execution using form/webhook  
* Real-time email confirmation (Gmail)  
* Data storage in Google Sheets  
* Input validation and error handling  
* Dynamic field mapping using n8n expressions  

---

## 🛠️ Tech Stack

* n8n  
* Google Sheets  
* Gmail  
* Webhooks  
* JSON  

---

## 🔄 How It Works

1. User submits hotel room booking form  
2. n8n workflow is triggered  
3. Input data is validated  
4. Booking details are stored in Google Sheets  
5. Confirmation email is sent to the user  
6. Workflow handles errors to ensure reliability
